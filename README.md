# Clasificador Web de Evidencias OVNI

Proyecto desarrollado para el examen recuperatorio de la materia Programación Web I.

## Descripción del Proyecto
Esta aplicación web funciona como un **Sistema de Clasificación de Evidencias OVNI**. Permite a los usuarios ingresar datos sobre avistamientos reportados y, mediante un algoritmo interno, clasifica la veracidad de la evidencia como "Débil", "Moderada" o "Fuerte" basándose en criterios técnicos como la existencia de material audiovisual, detección por radar y número de testigos.

## Funcionalidades
- **Registro de datos:** Formulario interactivo para capturar los detalles del fenómeno.
- **Análisis automático:** Clasificación instantánea del reporte mediante JavaScript.
- **Visualización dinámica:** Cambios en la interfaz (mensajes y colores) según el resultado del análisis.
- **Galería Multimedia:** Sección de archivos desclasificados (imágenes y videos) para dar contexto al proyecto.
- **Diseño Responsivo:** Adaptable a diferentes tamaños de pantalla mediante CSS.

## Tecnologías Utilizadas
- **HTML5:** Estructura semántica del contenido.
- **CSS3:** Diseño visual, estilo *Cyber-Glass* y uso de media queries para responsividad.
- **JavaScript:** Lógica de validación de campos, manipulación del DOM y cálculo de puntajes.
